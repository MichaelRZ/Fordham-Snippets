Hello World
===========
Output "Hello world!" to the console.

"Hello World" is a sanity check to make sure you haven't bricked your machine.

   .. code-tab:: c++ C++

          #include <iostream>
          using namespace std;

          int main(){
               cout<<"Hello world!"<< endl;
               return 0;
          }

   .. code-tab:: c# C#

          class Program

                  System.Console.WriteLine("Hello world!");
              }
          }
